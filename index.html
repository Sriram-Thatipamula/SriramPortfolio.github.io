Some of my Portfolio Projects.

Aviation Data Flow Management

Designed and implemented a scalable AWS-based ETL pipeline integrating real-time flight data with airport dimensions, leveraging S3, Glue, Redshift, and Step Functions to create a denormalized fact table for seamless analytics, achieving efficient data processing, enriched insights, and streamlined querying with robust error handling and automated notifications for operational reliability.

Accent Recognition Using MFCC

This project explores identifying regional accents within six languages. Using speech samples, it extracts features and trains machine learning models (Decision Trees, Random Forest, etc.) to classify speakers. This system aims to improve speech processing technology and user experience in voice-controlled systems and personalized language interfaces.

Automobile Spare Parts Sales Analysis

This project conducts an in-depth analysis of an automobile spares dataset, focusing on sales trends, client preferences, and warehouse performance. Utilizing data cleansing, exploratory data analysis, and visualization, the project addresses queries about sales distribution, product pricing, and quantity frequency, aiming to optimize inventory and pricing strategies.
